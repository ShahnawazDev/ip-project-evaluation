A Python program that calculates the following metro timings, travel times, and upcoming trains across the Blue Line (Line 3 & 4) and Magenta Line (Line 8) of the Delhi Metro.
It utilises structured time logic, dynamic peak–off–peak intervals, and station-to-station traversal, all based on data from a text file.

Features:-
 - Time Class :
    - Create and manipulate time objects
    - Add and subtract time
    - converts integers into time objects
 - Metro Schedule Logic :
    - Outputs metro departure times between 6:00 and 23:00
    - Between :
        - Peak hours frequency: Every 4 minutes (08:00 to 10:00) and (17:00 to 19:00)
        - Off-peak hours: Every 8 minutes (06:00 to 08:00), (10:00 to 17:00) and (19:00 to 23:00)
 - Travel Time Calculator: Calculates travel time between any two stations present on the blue and magenta lines.
 - Gives Prediction: on next arrival, direction-wise timing based on current time, source station and destination.
